# Proceso

Este índice agrupa las reglas y documentos operativos de la plantilla.

## Orden recomendado

1. Define `branching.md`.
2. Revisa `workflow.md`.
3. Ajusta `labels.md`.
4. Confirma `qa.md`.
5. Define `versioning.md`.
6. Aplica `branch_protection.md` en GitHub.
