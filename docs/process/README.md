# Proceso

Este índice agrupa las reglas y documentos operativos de la plantilla.

## Qué incluye

- [BRANCHING.md](./BRANCHING.md)
- [WORKFLOW.md](./WORKFLOW.md)
- [QA.md](./QA.md)
- [VERSIONING.md](./VERSIONING.md)
- [LABELS.md](./LABELS.md)
- [BRANCH_PROTECTION.md](./BRANCH_PROTECTION.md)

## Orden recomendado

1. Define `BRANCHING.md`.
2. Revisa `WORKFLOW.md`.
3. Ajusta `LABELS.md`.
4. Confirma `QA.md`.
5. Define `VERSIONING.md`.
6. Aplica `BRANCH_PROTECTION.md` en GitHub.

## Relación con otros archivos

- [`README.md`](../README.md) enlaza este índice para la visión general.

- [`SECURITY.md`](../SECURITY.md) sigue en la raíz porque GitHub lo reconoce como política de seguridad.
